# AI-classification-model using Teachable Machine 
 This AI model is  trained to classify pictures to 3 classes
 
- Cars
- planes
- boats
  
#training the model 
1. Open Google Teachable Machine
2. create an image project
3. add the three classes - Cars - planes - boats
4. upload 3~4 images for each class -the more the better-
5.  Train the model.
6. Export the trained model in TensorFlow → Keras format.
